## Astro Sonnet 📜

> A polished blog and portfolio starter built with Astro for content-driven sites.

### Get Started

```
pnpm dlx degit astrostarters/astro-sonnet my-blog

cd my-blog

pnpm install
pnpm dev
```

### Features

- 🧑‍🚀 **Astro v6** starter for a polished portfolio + blog
- ⚡ **React** integration ready (`@astrojs/react`)
- 📓 **MDX** blog system with typed content collections
- 🗃️ Built-in **blog** pages:
  - **Post** pages (`/blog/[slug]`)
  - **Blog** index (`/blog`)
  - **Tag** pages (`/blog/tags` and `/blog/tags/[tag]`)
- 🎨 **Tailwind CSS v4** preconfigured
- 🧰 **shadcn/ui** configured and ready to use
- 🌗 **Dark/light/system** theme toggle with saved preference
- 🔎 **SEO basics** included (`astro-seo`, `sitemap`, `robots.txt`)
- 📦 **RSS feed** for the blog posts
- ⚙️ TypeScript strict + path aliases (`@/*`)
- 🧼 Code quality tooling out of the box (**Biome/Ultracite** + **pre-commit hooks**)
